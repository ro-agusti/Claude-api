# Claude API Course

My notes and exercises from the official Anthropic course: [Claude with the Anthropic API](https://anthropic.skilljar.com/claude-with-the-anthropic-api).

## Requirements

- Python 3.x
- Anthropic API key

## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   cd your-repo-name
   ```

2. Install dependencies:
   ```bash
   pip3 install anthropic python-dotenv
   ```

3. Create a `.env` file in the root folder and add your API key:
   ```
   ANTHROPIC_API_KEY=your-api-key-here
   ```

Run any `.py` file directly:
```bash
python3 filename.py
```
